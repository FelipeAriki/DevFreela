# DevFreela

API Desenvolvida utilizando arquitetura limpa e padrões CQRS e Repository, com o intuito de gerenciar e auxiliar clientes para que seus projetos sejam desenvolvidos por freelancers com as skills necessárias e próprias.
Tecnologias utilizadas:
-Entity Framework Core;
-Fluent Validation;
-ASP.NET Core;
-.Net 8;
-MediatR;
-SQL Server;
-Notifications;
